# State-Building Ujamaa GIS Replication Workflow

This repository documents a reproducible ArcPy-based GIS workflow for constructing district-level spatial variables used in an economics research project on state-building and the Tanzanian Ujamaa policy.

The workflow reconstructs historical administrative boundaries, joins villagization treatment data, and generates spatial controls such as distance to Dar es Salaam, distance to the Uganda border, average elevation, average slope, colonial mission counts, historical road length, and interpolated rainfall measures.

The goal of this repository is not to reproduce the full econometric analysis, but to make the GIS preprocessing steps transparent, modular, and easier to audit or rerun.
